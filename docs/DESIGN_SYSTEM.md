# Aturan Baku Desain

## Tema warna

Project ini menggunakan kombinasi biru dan putih sebagai aturan baku.

Warna utama:

- Primary: `#1d4ed8`
- Primary dark: `#1e40af`
- Primary soft: `#dbeafe`
- Background: `#f3f8ff`
- Surface: `#ffffff`
- Text utama: `#0f172a`
- Text sekunder: `#475569`
- Border: `#cfe1f7`

## Aturan penggunaan

- Tombol aksi utama memakai biru primary dengan teks putih.
- Tombol sekunder memakai background putih, border biru muda, dan teks biru gelap.
- Panel dan card memakai putih dengan border biru muda.
- Badge status memakai biru soft.
- Hindari warna dominan selain biru-putih kecuali untuk status bahaya atau sukses.
- Warna merah hanya untuk aksi destruktif seperti batal atau hapus.
- Warna hijau hanya untuk pesan sukses.

## Layout

- Radius komponen maksimal 8px untuk panel, input, tabel, dan tombol.
- Halaman operasional harus langsung menampilkan fitur, bukan landing page marketing.
- Mobile harus tetap satu kolom dan tombol tidak boleh pecah teks.
- Tabel boleh horizontal scroll di mobile agar data tidak saling menimpa.

Aturan ini berlaku untuk seluruh halaman aplikasi, termasuk dashboard pendaftar dan konsol petugas.
