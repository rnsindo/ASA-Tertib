# Dokumentasi Project ASA-Tertib

ASA-Tertib adalah MVP aplikasi antrian universal berbasis Laravel, MySQL, Livewire, Socialite, dan Spatie Permission. Implementasi awal diarahkan untuk Pendaftaran Siswa Baru tingkat SMK dengan dua layanan utama: Verifikasi Berkas dan Wawancara.

## Dokumen

- `IMPLEMENTATION.md` - fitur dan struktur yang sudah diimplementasikan.
- `INSTALLATION.md` - kebutuhan sistem dan langkah instalasi baru.
- `DESIGN_SYSTEM.md` - aturan baku desain biru-putih.
- `PRESENCE_QUEUE_DESIGN.md` - rancangan sistem antrian wajib di tempat.
- `CHANGELOG.md` - catatan perubahan penting yang tidak boleh hilang.
